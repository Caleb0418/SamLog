담당 수업 일괄 등록
엑셀·CSV 파일 형식 안내
아래 두 가지 형식 중 편한 쪽을 사용하세요.
추천 담당 학급 템플릿 한 행에 여러 반을 묶어 입력
학년 담당 학급 과목 선생님 이름 비고
1학년 1반,2반,3반 국어 김국어
2학년 1반,2반 국어 김국어
1한 행이 여러 수업으로 자동 분리돼요. 위 예시는 5개 수업(1-1, 1-2, 1-3, 2-1, 2-2 국어)으로 등록됩니다.
2학급 구분자는 쉼표(,), 공백, 슬래시(/) 모두 인식해요. (예: 1반,2반 · 1 2 3 · 1/2/3)
3시간표 파일을 이미 올렸다면 요일·교시가 자동으로 매칭돼요. 한 과목이 주 3회면 3개 슬롯이 한 번에 채워집니다.
기본 표준 템플릿 한 행에 한 수업
학년 반 과목 요일 교시
11국어월1
12국어화3
23수학
1요일·교시를 직접 적으면 그대로 슬롯이 등록돼요.
2비워두면 시간표 파일에서 같은 과목을 찾아 자동 매칭됩니다.
3요일은 한글(월화수목금)·영어(mon~fri)·숫자(1~5) 모두 인식해요.
📌 공통 규칙
  • 헤더 행은 1행이 아니어도 돼요. 1·2행에 제목/안내가 있으면 자동으로 진짜 헤더를 찾아요.
  • 컬럼명은 일부만 일치해도 인식돼요. 예: 담당 학급 (쉼표 구분) → "담당 학급"으로 인식
  • 학년·반은 1학년처럼 단위가 붙어 있어도 숫자만 자동 추출해요.
  • 같은 (학년-반-과목)이 이미 등록돼 있으면 중복으로 표시되고 기본 미선택돼요.
